# tfmodule-aws-rds-aurora

## AWS RDS Aurora 사용을 위한 동적 Terraform 자동화 도구 

## Installation

```
git clone https://github.com/nuax79/tfmodule-aws-rds-aurora.git
```

## RDS Aurora for Build

```
terraform init
terraform plan
terraform apply
```
